This project is a simple calculator built using web technologies. 
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through a clean user interface. 
The project helps learners understand how HTML, CSS, and JavaScript work together in a real-world mini application. 
Tools used: 
HTML5 – to structure the calculator layout and buttons.
CSS3 – to design a responsive and user-friendly interface. 
JavaScript – to handle user input and perform calculations. 
VS Code – code editor. 
Key features: Simple calculator UI. 
Supports basic arithmetic operations (+, −, ×, ÷). 
Clear and delete functionality. 
Error handling for invalid expressions.
Responsive design (works on different screen sizes). 
Code separated into HTML, CSS, and JavaScript files. 
Steps to run it:  
Create a project folder named Calculator. 
Create three files:  
index.html 
style.css 
script.js 
Write the calculator structure in HTML 
Style the calculator using CSS for a clean look 
Add JavaScript functions to handle button clicks and calculations 
Link CSS and JavaScript files to the HTML file 
To run it:
Open the folder in VS Code 
Right-click index.html 
Select Open with Live Server
